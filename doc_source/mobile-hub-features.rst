##################
|AMHlong| Features
##################


.. meta::
   :description: Getting Started with |AMHlong|.


|AMHlong| is a service that enables even a novice to easily deploy and configure mobile app backend
features using a range of powerful AWS services.

You create a free project, then choose and configure mobile app features using a point and click
console. |AMH| takes care of the complexities in the background and then supplies you with step by
step integration instructions.
